What is ReVens?
ReVens is a Windows-based Reverse Engineering Toolkit "All-In-One", Built for Security (Penetration testing) & Educational purposes only.

Warning:
Reverse Engineering tools are denied by Antivirus (Due to binary patching algorithms, debugging ...etc), You should make an exception to avoid detection, or use a secured virtual machine.

Source: https://github.com/Jakiboy/ReVens
